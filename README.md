# ⌨️ Speed Typing Test (Python)

## 📌 Overview
A command-line Speed Typing Test application developed in Python.  
The program measures a user's typing speed and accuracy by comparing typed input against a reference text.

This project was created during internship practice to strengthen Python fundamentals and real-time input handling.

---

## ✨ Features
- ⏱️ Typing speed calculation (WPM)  
- 🎯 Accuracy measurement  
- 📝 Real-time user input  
- 💻 Lightweight CLI application  
- 📊 Performance feedback  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Concepts Used:** Time module, String comparison, User input handling
